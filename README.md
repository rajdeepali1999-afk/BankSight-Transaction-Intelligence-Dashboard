# BankSight: Transaction Intelligence Dashboard

## Overview
BankSight is a banking analytics dashboard built using Python, SQL, and Streamlit.

## Features
- Customer & transaction analysis
- Fraud detection
- 15+ SQL queries
- Interactive dashboard
- CRUD operations

## Tech Stack
- Python
- SQL (SQLite)
- Streamlit
- Plotly

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py
